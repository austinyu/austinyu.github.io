  • Let P and Q be arbitrary bollean values.

  • In this problem, we need to show the two statements “if P then Q” and “if not Q then not P” are logically equivalent. In other words, we show both if the statement “if P then Q” is true then the statement “if not Q then not P” is true and if “if P then Q” is false then the statement “if not Q then not P” is false.

  • We first show if the statement “if P then Q” is true then the statement “if not Q then not P” is true.

    • Given the statement “if P then Q” is true, we have three cases that make the statement true. The three cases are bellow:

      • P Q
        True True
        False True
        Falsee False

    • Case 1: P is true and Q is true, then “if P then Q” is true. We need to show “if not Q then not P” is true

      Since P is true and Q is true, we know (not P) and (not Q) are false. So the statement “if not Q then not P” is true

    • Case 2: P is false and Q is true, then “if P then Q” is true. We need to show “if not Q then not P” is true

      Since P is false and Q is true, we know (not P) is true and (not Q) is false. So the statement “if not Q then not P” is true

    • Case 3: P is false and Q is false, then “if P then Q” is true. We need to show “if not Q then not P” is true

      Since P is false and Q is false, we know (not P) is true and (not Q) is true. So the statement “if not Q then not P” is true

    • We have shown the three cases exhaust all possibilities of P and Q, so we conclude that “if P then Q” is true implies the statement “if not Q then not P” is true unconditionally.

  • We next show if the statement “if P then Q” is false then the statement “if not Q then not P” is false.

    • Given the statement “if P then Q” is false, we only have one case: P is true and Q is false.

    • P is true and Q is false, then “if P then Q” is false. We need to show “if not Q then not P” is false

      Since P is true and Q is false, we know (not P) is false and (not Q) is true. So the statement “if not Q then not P” is false

    • We have now shown if the statement “if P then Q” is false then the statement “if not Q then not P” is false.

  • Since we have shown both if the statement “if P then Q” is true then the statement “if not Q then not P” is true and if “if P then Q” is false then the statement “if not Q then not P” is false, we conclude that “if P then Q” and “if not Q then not P” are logically equivalent. In other words, “if P then Q” statements and its contrapositive is logically equivalent.
