Advanced Lists

  • taking elements

    Scheme also provides two other operations to extract values from lists: car extracts the first element (element zero) and cdr returns a list containing all but the first element.

    ​ 

    • > (car '(2 3 4 5))
2
> (cdr '(2 3 4 5))
'(3 4 5)
> (list-ref '(2 3 4 5 6) 2)
4

> (define rainbow (list "red" "orange" "yellow" "green" "blue" "indigo" "violet"))
> (cadr rainbow)
"orange"
> (cddr rainbow)
'("yellow" "green" "blue" "indigo" "violet")
> (caddr rainbow)
"yellow"

Local Bindings

  • let 

    • (let
  ([name1 exp1]

  bodym)

(define midwest-central-longitude
  (lambda (cities)
    (let ([sorted-midwest-cities
           (sort (filter midwest-state? cities)
                 (lambda (entry1 entry2)
                   (< (caddr entry1) (caddr entry2))))])
      (* 1/2
         (+ (caddr (car sorted-midwest-cities))
            (caddr (car (reverse sorted-midwest-cities))))))))

  • Let procedure

    • (define hypotenuse-of-right-triangle
  (let ([square (lambda (n) (* n n))])
    (lambda (first-leg second-leg)
      (sqrt (+ (square first-leg) (square second-leg))))))

Varyfying preconditions

  • huscks kernels

    • (define northernmost-latitude
  (lambda (cities)
    (when (or (not (list? cities))
              (null? cities)
              (not (all-cities? cities)))
      (error "northernmost-latitude: expects a non-empty list of cities, given "
             cities))
    (northernmost-latitude-kernel cities)))

(define northernmost-latitude-kernel
  (lambda (cities)
    (reduce max (map cadr cities))))

  • Boolean recursion

    • (define all-numbers?
  (lambda (values)
    (or (null? values)
        (and (number? (car values))
             (all-numbers? (cdr values))))))

(define any-numbers?
  (lambda (values)
    (and (not (null? values))
         (or (number? (car values))
             (any-numbers? (cdr values))))))

  • List recursion

    • (define recursive-proc
  (lambda (lst)
    (if (null? lst)
        (base-case)
        (combine (car lst)
                 (recursive-proc (cdr lst))))))
