Advanced Lists

  • taking elements

    Scheme also provides two other operations to extract values from lists: car extracts the first element (element zero) and cdr returns a list containing all but the first element.

    ​ 

    • > (car '(2 3 4 5))

> (cdr '(2 3 4 5))
'(3 4 5)
> (list-ref '(2 3 4 5 6) 2)

  (lambda (cities)

Varyfying preconditions

  • huscks kernels

    • (define northernmost-latitude
  (lambda (cities)
    (when (or (not (list? cities))
              (null? cities)
              (not (all-cities? cities)))
      (error "northernmost-latitude: expects a non-empty list of cities, given "
             cities))
    (northernmost-latitude-kernel cities)))

(define northernmost-latitude-kernel
  (lambda (cities)
    (reduce max (map cadr cities))))

  • Boolean recursion

    • (define all-numbers?
  (lambda (values)
    (or (null? values)
        (and (number? (car values))
             (all-numbers? (cdr values))))))

(define any-numbers?
  (lambda (values)
    (and (not (null? values))
         (or (number? (car values))
             (any-numbers? (cdr values))))))

  • List recursion

    • (define recursive-proc
  (lambda (lst)
    (if (null? lst)

        (combine (car lst)
                 (recursive-proc (cdr lst))))))

  • paradigms 

    • (define vector-proc
  (lambda (vec other)
    (let ([len (vector-length vec)])
      (let kernel ([pos 0])
        (if (= pos len)
            (base-case vec other)
            (combine (vector-ref vec pos)
                     (kernel (+ pos 1))))))))

(define vector-proc
  (lambda (vec other)
    (let kernel ([pos (- (vector-length vec) 1)])
      (if (< pos 0)
          (base-case vec other)
          (combine (vector-ref vec pos)
                   (kernel (- pos 1)))))))

(define vector-proc
  (lambda (vec other)
    (let ([len (vector-length vec)])
      (let kernel! ([pos 0])
        (cond
          [(= pos len)
           (base-case! vec other)
           ...]
          [else
           (operation! vec pos)
           ...
           (kernel! (+ pos 1))])))))

Binary Tree

  • Paradigm 

    • (define tree-proc
  (lambda (tree)
      (if (empty? tree)
          (base-case other)
          (combine (contents tree)
                   (tree-proc (left tree))
                   (tree-proc (right tree))))))
